# Component Inventory: Aki's World (Wave 0)

## Atoms (Foundations)
- **Typography**: Sora (Headings), Inter (Body), Roboto Mono (Data/Timestamps).
- **Colors**: Pink `#FF6CA4`, Bubblegum `#FFD6EC`, Peach `#FFF2F7`.
- **Buttons**:
    - `Primary`: Pink background, scale on hover.
    - `Secondary`: Bubblegum background, dark text.
    - `Ghost`: Transparent, pink text.
- **Inputs**: Rounded `1.25rem`, pale peach background, pink focus ring.
- **Chips/Tags**: Small rounded bubbles for emotions/tags.

## Molecules (Simple Components)
- **PlushieCard**: 3D tilt effect, image, name, trauma description.
- **StoryCard**: "Torn paper" edge effect, title, excerpt, date.
- **ConfessionCard**: Sticky note aesthetic, rotation variation.
- **Microinteractions**:
    - Heart reaction (particle burst).
    - Sticker drop (drag and drop).
- **Toast**: Floating "bubble" notification from bottom-center.

## Organisms (Complex Sections)
- **Header**: Sticky "bubble" nav with plushie avatar.
- **Footer**: Glassmorphism, social badges, "made with insomnia".
- **IntroSequence**: Typing animation container + 3D Hero canvas.
- **AkiUploadWall**: Owner dashboard with drag-drop zone.
- **AdminQueue**: List of confessions with moderation actions.
- **AudioPlayer**: Persistent bottom bar with waveform visualization.

## Templates (Layouts)
- **PageShell**: Global wrapper with responsive padding.
- **AuthGate**: Password entry focused layout.
- **Grid**: Responsive masonry-like grid for plushies/stories.
