# Aki's World ✨

> plushies, 03:14 confessions & travel scars — posted honestly

The digital messy-cute sanctuary of Aki.

## Tech Stack
- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS + Design Tokens
- **Motion:** Framer Motion + Three.js
- **Data:** Firestore (planned) + Vercel Blob

## Getting Started

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Environment Setup:**
    Copy `.env.local.example` to `.env.local`:
    ```bash
    cp .env.local.example .env.local
    ```

3.  **Run Development Server:**
    ```bash
    npm run dev
    ```
    Open [http://localhost:3000](http://localhost:3000)

## Design System
- **Fonts:** Sora (Headings), Inter (Body), Roboto Mono (Mono)
- **Colors:** Pink `#FF6CA4`, Bubblegum `#FFD6EC`, Peach `#FFF2F7`
- **Radius:** `1.25rem` (20px)
